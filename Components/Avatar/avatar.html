Avatar

An avatar is a graphical representation of a user or the user's character, they are user profile pictures, available in circular shapes. Avatars can be used for 'container' objects — projects, spaces, repositories, etc — to give them a visual identity.

Profile Avatar

Profile Avatars can be created using the class 'avatar', inside the img tag.

Sized Avatars

Size of the avatars can be changed by adding avatar-sm | avatar-md | avatar-lg class to the existing avatar class.

Square Avatars

Adding a class 'avatar-square' to existing class, will generate a square shaped avatar.

Note : Square class wouldn't work if the image provided is svg.
